Answer

 

Hi Ram,

 

 

It’s great to know that you have chosen Windows 8 and thank you for providing us this opportunity to assist you on Microsoft Community Forums.

 

 

You would like to know if there are any classic themes available for Windows 8 which allows the user to use the computer very fast when compared to the other themes.

 

 

Unfortunately, there is no "Windows Classic Theme" that is available with the new Windows 8 operating system.

 

You can refer to these links to know on how to improve the performance of your PC and the tips to save the battery power on Windows 8.

 

http://windows.microsoft.com/en-US/windows-8/improve-pc-performance

To know about the Windows 8 themes, you can have a look at this link:

 

http://windows.microsoft.com/en-US/windows/themes

 

Hope this information is helpful. Please feel free to get back to us for any further queries or issues related to Windows.
